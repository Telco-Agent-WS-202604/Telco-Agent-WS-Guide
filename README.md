# Telco DevAgent Workshop

通信事業者向けネットワークインフラ運用を AI エージェント（Strands Agents SDK + Amazon Bedrock AgentCore）で自動化するハンズオンワークショップです。

SOP（Standard Operating Procedure: 標準作業手順書）とチャットによる自然言語指示を組み合わせ、AI エージェントがネットワーク機器やアプリケーションに自律的にアクセスして、調査・設計・構築・検証を行います。

## シナリオ

本ワークショップでは 2 つのシナリオを体験します。両シナリオとも同じ 1 つの AI エージェントが、シナリオに応じて自律的に適切な SOP・ツールを選んで実行します。

### 1. Router Config — ISP バックボーンの冗長性監査と障害是正

6 台の FRR ルーターで構成された ISP バックボーンネットワークに対して、AI エージェントがトポロジ調査・ヘルスチェック・隠れた冗長性欠如の発見・iBGP 障害是正・監査レポート生成を実施します。

大規模ネットワーク運用において、人手では見落としがちな潜在的な障害リスクを AI が網羅的に検出・是正する価値を体験できます。

**[workshop-guide-router.md](./workshop-guide-router.md)**

### 2. UPF Deployment — 5G コアネットワークへのスライス追加

稼働中の 5G コアネットワーク（free5GC）に新しいネットワークスライスを追加します。AI エージェントが既存設定ファイル分析・パラメータ設計・設定変更・GitHub 経由での CI/CD デプロイ・ヘルスチェック・作業レポート生成までを一貫して実行します。

複数アプリケーションにまたがる整合性を保った設定変更と、既存サービス無影響でのインフラ拡張を AI が自律的に行う価値を体験できます。

**[workshop-guide-upf.md](./workshop-guide-upf.md)**

## 進め方

各シナリオは独立しているため、どちらから始めても問題ありません。UPF シナリオの Step 3（デプロイ）では CI/CD の完了を 10〜15 分待つ時間が発生するため、その待ち時間で Router シナリオを進める構成がおすすめです。
